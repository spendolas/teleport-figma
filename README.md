# Teleport

A Figma plugin that sends selected layers to a dedicated section on another page in the same file, with automatic layout formatting.

## Install

1. Download `teleport.zip` from the [latest release](../../releases/latest)
2. Unzip the archive
3. In Figma, go to **Plugins > Development > Import plugin from manifest...**
4. Select the `manifest.json` file from the unzipped folder

## Usage

1. Select one or more layers on the current page
2. Run **Teleport** from the Plugins menu
3. Choose a target page
4. Pick an existing Teleport section or create a new one
5. Set the layout mode (Horizontal, Vertical, or Grid) and spacing
6. Click **Teleport** to move the originals, or **Copy** to duplicate them
